A tutorial to Boa

Types

Boa has a variety of types to use in your code. These different types are utilized for varying tasks that may be needed in your code.

The following types are available:

  • NUMBER
  • STRING
  • LIST
  • MAP

NUMBER

The NUMBER type is just literally a double. The NUMBER type has access to all operators except for the 'get' (alias '?') operator. Please note that the NUMBER type is the only type to use the '^' operator.

>>> 1
1.000000

STRING

The STRING type is a string of characters. The STRING type has access to all operators except for the '-' and '/' operators.

>>> "Hello World!"
Hello World!

LIST

The LIST type is a list of any type of values. The LIST type has access to all operators except the '/' operator.

>>> [1, 2, ["Nested List!", 1], {}]
[1.000000, 2.000000, [Nested List!, 1.000000], {}]

MAP

MAPs store multiple key value pairs of any type. The MAP type has access to all operators except the '/' and '*' operator.

>>> {"key1": 1, 2: "key2", "key3": ["Nested List!", 1]
{key1: 1.000000, 2.000000: key2, key3: [Nested List!, 1.000000]}

Operators

Boa has a variety of operators to use in your code.

Expressions

Boa allows you to perform many forms of expressions. An expression is a combination of values and operators.

A simple math expression

>>> 1 + 2
3.000000
>>> 40 - 3 * 2
34.000000
>>> (40 - 3) * 2
74.000000

Please note that this language is stupid and a NUMBER is just a double, no ints or floats :( Also note that I'm too lazy to show examples but there's the '^' operator, '/' operator, and '%' operator.

Expressions with strings and lists

>>> "string1" + "string2"
string1string2
>>> "word" * 3
wordwordword
>>> "hello" ? 0 // Returns the 0th index of the string
h
>>> [1, "2", 3] + 4 // Pushes back the value 4 to the list
[1.000000, 2, 3.000000, 4.000000]
>>> [1, 2, 3] - 2 // Removes the 2nd index of the list
[1.000000, 2.000000]
>>> [1, 2, 3] * [4, "fifth", 6] // Combines two lists
[1.000000, 2.000000, 3.000000, 4.000000, fifth, 6.000000]
>>> [1, 2, 3] ? 1 // Returns the 1st index of the list
2.000000
